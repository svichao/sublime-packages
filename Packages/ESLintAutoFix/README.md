# ESLintAutoFix

A sublime plugin. 

Use global eslint command.

## usage

You should modify the setting first before use.

## setting

### eslint_path

* Global installation path of eslint, default '/usr/local/bin/'.

### show_panel

* Whether the output panel, default true.

e.g.

```
{
  "eslint_path": "/usr/local/bin/",
  "show_panel": true
}

```

## command

### `eslint_auto_fix`

## keymap

### `super+shift+,`
